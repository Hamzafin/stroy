Заявка

Менеджеры подтягиваются автоматически из тегов.
ЗП — считает по заявкам (10т=1500, 20т=2500). Другое — ручной расход.
Выбери менеджера — и контрагенты подтянутся по его тегу.
При клике показывает список. При вводе — фильтрует.
Позиции: 0
Пробелы ставятся автоматически: 300 000
Выбери контрагента из списка — долг подтянется автоматически.
Удаление заявки здесь удаляет её из базы бота (для перерасчёта зарплаты). Сообщение в чате поставщика не удаляется.
Нажми “Отправить в бот” — уйдёт расход “ЗП” с суммой (ручной если заполнена, иначе по расчёту).
Комментарий можно добавить ниже.
